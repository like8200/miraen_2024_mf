icon & images guide - 디자인 반영 전 기본 마크업 샘플 입니다.

default 초/중/고 모두 Icon은 svg를 기본으로 사용하고, Images는 png를 기본으로 사용합니다.

  • /assets/images/common : 초/중/고 공용 icon 및 images
    /assets/images/elementary : 초등 icon 및 images
    /assets/images/middle : 중학 icon 및 images
    /assets/images/high : 고등 icon 및 images
    /assets/images/temp : 샘플, dummy icon 및 images
  • icon명은 icon-xxx.svg, images명은 img-xxx.png를 기본으로 사용합니다.
    ※ logo는 제외 logo-xxx.svg
    ※ background-image는 제외 bg-xxx.svg 또는 bg-xxx.png
  • icon명은 디자인 그룹 명을 기본으로 사용하나 중복 명은 font awesome을 기준으로 합니다.
    ※ footer icon은 예외
  • images 저장 시 2배를 기본으로 사용합니다.
