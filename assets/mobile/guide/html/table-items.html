table-items

list type thead + tbody 로 사용시 기본 중앙정렬 되며, 금액은 우측 정렬 사용합니다.

  • summary는 삭제 된 속성으로 caption을 사용합니다. table 바로 아래에 caption을 사용해주세요.
  • 세로로 그룹핑되는 리스트형 테이블인 경우 scope="col"
  • colspan 사용시 scope="colgroup"
  • 가로로 그룹핑되는 상세보기형 테이블인 경우 scope="row"
  • rowspan 사용시 scope="row"
  • tbody안의 th는 왼쪽 정렬 및 컬러 black
  • 마우스 호버 시 tr 하이라이팅(링크 여부에 따라 밑줄 그어짐)
